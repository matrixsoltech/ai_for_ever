## 🚀 Getting Started

### Prerequisites

- Python 3.8+
- Git

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/matrixsoltech/ai_for_ever.git
   ```

2. Navigate to project directory
   ```bash
   cd ai_for_ever
   ```

3. Install dependencies (once requirements.txt is set up)
   ```bash
   pip install -r requirements.txt
   ```
